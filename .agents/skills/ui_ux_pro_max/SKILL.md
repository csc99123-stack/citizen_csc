# Skill: UI/UX Pro Max

## Purpose
Sets the standard for building beautiful, responsive, and highly interactive user interfaces.

## Guidelines
- Use Tailwind CSS heavily for styling. Never write custom `.css` files unless overriding a deeply entrenched third-party package style.
- Employ Shadcn UI components for accessible, customizable native elements.
- Implement 'glassmorphism', smooth gradients, and subtle framer-motion micro-animations where it enhances user experience.
- Maintain consistent padding, margins, and typography scales (e.g., using `font-sans` with an imported font like Inter or Outfit).
- Build fully responsive designs (Mobile-first approaches for B2B dashboards, but strictly accessible on smaller devices).
